# Algorithmic Trading Backtesting Engine

This project implements a Python-based backtesting engine for algorithmic trading strategies.

## Strategies Implemented
- Moving Average Crossover
- Momentum Strategy
- Mean Reversion Strategy

## Metrics
- Sharpe Ratio
- Maximum Drawdown
- Portfolio Returns
- Equity Curve

## Technologies
Python, Pandas, NumPy, Matplotlib

## How to Run
pip install -r requirements.txt
python download_data.py
python main.py# algorithmic-trading-backtesting-engine
